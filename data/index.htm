WiFi relay

This is an example of the operation of the ESP8266 as a relay control device.

The relay can be controlled on the Remote page
Modifying with a sketch you can control more than two relays.

Supports SSDP and WebUpdate
The device is visible in the networked Windows environment. You can update the firmware via the Web interface from the File Upload page.

Connects to home WI-FI network
Combines all devices in the apartment.
